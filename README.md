## ⚙️ Installation and Setup

1. Clone the repository

git clone https://github.com/shauryafrontend-droid/Portfolio-Website-.git

2. Navigate to the project folder

cd Portfolio-Website-

3. Install dependencies

npm install

4. Run the project

npm run dev
